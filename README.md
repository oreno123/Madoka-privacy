# Madoka 浏览器插件隐私政策

生效日期：2026年5月7日

## 1. 数据收集与使用说明
Madoka 浏览器插件仅在用户主动触发操作时，临时读取当前网页内容、用户选中的文本、或用户主动输入的对话内容，用于以下场景：
- 网页内容 AI 总结、阅读解析
- 划词翻译、网页引用对话
- 用户主动发起的联网搜索与 AI 交互

本插件**不会在后台静默收集、存储、上传**用户的个人身份信息、浏览历史、账号密码等敏感数据。

## 2. 数据存储与传输
- 所有临时读取的网页内容、用户文本仅用于当前会话的 AI 处理，不会永久存储或用于广告推送、第三方共享、商业售卖。
- 插件仅加载公共开源 CDN 依赖库（如 unpkg.com），以及用户主动触发的 AI 接口请求，不会后台静默联网上传用户数据。
- 插件配置、对话记忆等用户数据默认保存在**浏览器本地**，用户可主动开启 Obsidian 同步功能（同步数据由用户自行管理，插件不存储同步内容）。

## 3. 第三方服务
插件可能调用以下第三方服务，相关数据处理遵循对应第三方隐私条款：
- 公共翻译接口
- 用户配置的 AI 大模型接口
- 公共 CDN 资源加载服务

插件不会向第三方共享用户的个人身份信息。

## 4. 未成年人保护
本工具面向普通网页阅读与学习辅助场景使用，不刻意收集未成年人信息。若未成年人需使用，请在监护人指导下进行。
